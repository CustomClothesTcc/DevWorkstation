<h1>Empresa Tech - TCC 🚀</h1>

<p>Este repositório contém o código-fonte de um site fictício de uma empresa de tecnologia, desenvolvido como parte do Trabalho de Conclusão de Curso (TCC). O site foi projetado utilizando <strong>SCSS</strong> para estilos e <strong>JavaScript</strong> para interatividade.</p>

<h2>Estrutura de Diretórios 🗂️</h2>

<ul>
  <li><strong>img/</strong>
    <ul>
      <li>📷 Contém todas as imagens utilizadas no site.</li>
    </ul>
  </li>
  <li><strong>javascript/</strong>
    <ul>
      <li>📜 Contém os arquivos JavaScript responsáveis pela lógica e interatividade do site.</li>
    </ul>
  </li>
  <li><strong>styles/</strong>
    <ul>
      <li>🎨 Contém os arquivos SCSS utilizados para estilização do site.</li>
    </ul>
  </li>
  <li><strong>index.html</strong>
    <ul>
      <li>📄 Arquivo principal do site que estrutura o conteúdo e integra os estilos e scripts.</li>
    </ul>
  </li>
  <li><strong>package-lock.json</strong> e <strong>package.json</strong>
    <ul>
      <li>📦 Arquivos de configuração do projeto, gerenciados pelo npm.</li>
    </ul>
  </li>
</ul>

<h2>Tecnologias Utilizadas 🛠️</h2>

<ul>
  <li>🌐 <strong>HTML5</strong></li>
  <li>🎨 <strong>SCSS</strong></li>
  <li>⚙️ <strong>JavaScript</strong></li>
</ul>

<h2>Como Executar o Projeto ▶️</h2>

<ol>
  <li>Clone este repositório:
    <pre><code>git clone https://github.com/CustomClothesTcc/DevWorkstation.git</code></pre>
  </li>
  <li>Acesse o diretório do projeto:
    <pre><code>cd DevWorkstation</code></pre>
  </li>
  <li>
    <pre>Execute o arquivo Index.html 💻</pre>
  </li>
</ol>

<h2>Funcionalidades do Site 🌟</h2>

<ul>
  <li>🏠 Página inicial com apresentação da empresa.</li>
  <li>💼 Seção de serviços e soluções tecnológicas.</li>
</ul>
